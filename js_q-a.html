Q.47 What is the drawback of declaring methods directly in JavaScript objects?

=>

  • When you define a method directly within an object, it becomes specific to that object instance.
  • Other instances of the same object type won’t share the same method. This lack of reusability can lead to code duplication.
  • In contrast, defining methods on the prototype allows all instances of the same type to share the same method.
